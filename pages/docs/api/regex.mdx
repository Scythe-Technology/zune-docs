#### new

```luau
regex.new(pattern: string, flags: string?): Regex
```
<Callout type="warning" emoji="️⚠️">Can Error</Callout>

Compiles a regular expression pattern.

Flags:
    - `i`: Case insensitive
    - `m`: Multiline

##### Parameters
- `pattern: string` - The pattern to compile.
##### Returns
- [`Regex`](#regex) - The compiled regular expression.
##### Throws
- If the pattern is invalid, an error will be thrown.
